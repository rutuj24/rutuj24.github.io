OUTCOMES OF THE COURSE

  1. TECHNOLOGIES TAUGHT

    You will learn HTML, CSS, JS(Advanced), Bootstrap and Git by the end of this course. All of these are commonly used front end technologies.

  2. RESPONSIVE WEBPAGES

    Make resposnsive web pages, i.e pages which change according to the device.

  3. AESTHETIC DESIGNS

    Learn to add animations to the website to make it more interesting. These will be covered in advances CSS and JS.

  4. DEPLOYMENT

    Learn how to deploy your pages using git and Github. Also laern to buy domain for your website.
